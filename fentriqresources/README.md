# Fentriq — Brand Kit

Símbolo: **Folded Band** — una cinta de planos plegados que representa el stack completo
(front, back, Web3, IA, data) plegado en una sola pieza con impulso. Estudio de software full-stack.

## Archivos

**Logo (SVG, texto en curvas — no necesita la fuente instalada)**
- `fentriq-primary.svg` — horizontal, color. Uso general sobre fondo claro.
- `fentriq-stacked.svg` — apilado (espacios cuadrados/verticales).
- `fentriq-reverse.svg` — para fondo oscuro.
- `fentriq-mono-dark.svg` / `fentriq-mono-white.svg` — una sola tinta (sellos, B/N).
- `fentriq-mark.svg` / `fentriq-mark-mono.svg` — solo el símbolo.

**Íconos**
- `fentriq-icon-dark.svg` / `fentriq-icon-accent.svg` — squircle (app / avatar redes).
- `fentriq-icon-maskable.svg` — full-bleed para Android adaptive icons.

**Favicons / web**
- `favicon.ico` (16/32/48), `favicon-16.png`, `favicon-32.png`, `favicon-48.png`
- `apple-touch-icon.png` (180), `icon-512.png`, `fentriq-icon-maskable-512.png`
- `site.webmanifest`

**PNG de respaldo (transparentes):** `fentriq-primary.png`, `fentriq-reverse.png`,
`fentriq-stacked.png`, `*-512.png`.

## Colores
- Tinta: `#15171C`
- Gris facetado: `#3A3F4C` (claro: `#AEB4C2`)
- Acento índigo: `#4F5BF7`
- Acento violeta: `#8A6CFF`

## Tipografía
- Wordmark: **Space Grotesk** SemiBold. Para textos de marca usá Space Grotesk
  (Regular/Medium para cuerpo, SemiBold/Bold para títulos).

## Snippet para el `<head>` de la web

```html
<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">
<meta name="theme-color" content="#15171C">
```

## Uso
- Dejá aire alrededor del logo (mínimo = la altura del símbolo).
- No deformes, no cambies los colores de los facetados ni rotes el símbolo.
- Sobre fondos oscuros usá la versión reverse o mono-white.
